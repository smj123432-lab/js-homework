# js-homework

## 사민재 자바스크립트 과제

### 과제 수행 과정

먼저 과제를 수행하기 전에 어떤 방식으로 과제를 작성해야하는지 고민을 좀 해봤습니다.

1. 각각의 컴포넌트(1, 2, 3페이지)를 만들어 모달 다이얼로그 처럼 겹쳐주고 클릭 시 z-index를 조정해서 구현 해야하는가?

2. 하나의 컴포넌트만 만들어 그 안에 있는 사진과 설명만 클릭 시 바꿔주어야 하는가?

첫 번째 방법으로 하려니 한 번 수정할때 많은 것을 수정해야할 것 같아 유지보수 측면에서 좋지않다고 판단 됐습니다.

그래서 두 번째 방법으로 하려니 이해가 너무 안돼서 나중에 다시 고쳐두더라도 혼자 방법을 고민이라도 해볼 수 있는 첫 번째 방법을 써보려 했습니다.

그래서 첫번째 방법대로 하기 위해 섹션들을 각각 만들어보았으나 계속해서 css class명 수정하느라 시간이 다 갔습니다.

그러다보니 하면 할 수록 아닌 것 같다는 생각이 들어 최대한 ai를 안쓰려 했으나 결국 제미나이에게 도움을 받았습니다.

그러니 제미나이가 textcontent라는 방법을 추천해주었습니다. 그냥 무작정 따라하는 것보다는 이게 뭔지라도 알아야 이해를 하겠다 싶어 텍스트 콘텐트가 뭔지부터 알아보고 마크업부터 수정을 해주었습니다.
원래

```
<섹션>
  <헤더>
  </헤더>
  <메인>
    <아티클 크로노>
    </아티클 크로노>
    <아티클 아크에이지>
    </아티클 아크에이지>
  </메인>
  <푸터>
  </푸터>
</섹션>
```

이런 구조였던 것을

```
<섹션>
<헤더>
</헤더>
.
.
.
<푸터>
</푸터>
```

css에서 문제는 사진 부분에 문제가 생겼습니다.

- 캐릭터 이미지는 마크업으로 배경이미지는 css로 넣어서인지 배경이미지를 캐릭터보다 작게하면 위쪽이 동그래지지 않았고, 캐릭터 이미지에 맞게 키우니 동그래지긴 했지만 시안처럼 캐릭터으 ㅣ머리가 튀어나오지 않았습니다.
- 이를 해결해보려고 ai를 붙잡고 늘어졌지만 결국 해결하지 못했습니다.

와 같은 구조로 바꾸었습니다.
다만 뭔지 찾아보아도 이걸 어떻게 js로 연결해야하는지 잘 몰랐습니다.

제미나이는 게임 데이터들을 따로 마크업에 짜지 않고

1. 자바스크립트 배열로 게임 데이터들을 모아두고
2. 페이지 수를 정해두고
3. 게임 목록이 들어갈 곳과 버튼 들을 지정해서
4. render라는 화면 그리는 함수와 html코드를 넣어주는 함수를 사용하여 섹션 메인 안에 html 코드를 넣어주었습니다.
5. 그리고 textcontent로 번호를 업데이트 시켜줬습니다
6. 마지막으로 이전 버튼을 클릭하면 이전으로 다음 버튼을 클릭하면 다음으로 넘어가도록 함수를 짰습니다.

최대한 혼자 해보고자 하였으나 결국 ai의 도움을 너무 많이 받고 과제를 완성했습니다.
css에 대한 이해도도 너무 부족하고 js에 대한 이해도도 너무 부족하다는 생각이 들었습니다.

  몇 번을 경로수정하고 폴더 위치를 바꿔 다시 실행 해봐도 사진이 뜨지 않습니다
